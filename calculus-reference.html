CALCULUS EXAM REFERENCE

 

UNIT 1: LIMITS

 

Reading Limits from Graphs

Look at y-value as x approaches target. Both sides same → limit exists. Different → DNE.

Ex: limx→3 h(x) = −2
 

Direct Substitution

Plug in the value. If you get a number, done.

Ex: limx→−3 z²−6z+53z+6 = 32−3
 

Indeterminate Forms (0/0)

Factor and cancel, or simplify algebraically

Ex: limx→2 x²−4x−2 = limx→2 (x+2) = 4
 

Limits at Infinity (Polynomials)

Compare highest power: deg(top) > deg(bottom) → ±∞ | deg(top) = deg(bottom) → ratio of coefficients | deg(top) < deg(bottom) → 0

Ex: limx→∞ 4x³+6x−18x³+16x+21 = 41 = 4
 

Exponentials at Infinity

e−∞ = 0, e∞ = ∞

Ex: f(x) = 5e−x/25, limx→∞ f(x) = 0
 

Continuity

Need: (1) f(c) exists (2) limx→c f(x) exists (3) limx→c f(x) = f(c)

Piecewise: Set pieces equal at boundary, solve for k

 

Average Rate of Change

mavg = f(b)−f(a)b−a
 
 

UNIT 2: DERIVATIVES

 

Limit Definition

f'(x) = limh→0 f(x+h)−f(x)h

Steps: (1) Find f(x+h) (2) Subtract f(x) (3) Divide by h (4) Simplify/cancel h (5) lim as h→0

Ex: f(x) = √(x+1) → f'(x) = 12√(x+1)
 

Power Rule

ddx[xn] = nxn−1
 

Product Rule

ddx[fg] = f'g + fg'
Ex: 2cos(x)sin(x) → −2sin²(x) + 2cos²(x)
 

Quotient Rule

ddx[fg] = f'g−fg'g²

"Low d-high minus high d-low, square below"

 

Chain Rule

ddx[f(g(x))] = f'(g(x))·g'(x)
Ex: (3x+1)⁵ → 5(3x+1)⁴·3 = 15(3x+1)⁴
 

Common Derivatives

sin(x) → cos(x) | cos(x) → −sin(x) | tan(x) → sec²(x) | ex → ex | ln(x) → 1x

ef(x) → ef(x)·f'(x) | ln(f(x)) → f'(x)f(x) | ax → axln(a)

 

Logarithmic Differentiation

For y = [stuff][stuff]: ln(y) = ln(RHS), simplify, differentiate, y'y = result, y' = y·result

Ex: y = (5−4x³)x → ln(y) = xln(5−4x³) → y' = (5−4x³)x[ln(5−4x³) − 12x³5−4x³]
 

Implicit Differentiation

(1) Differentiate both sides w.r.t. x (2) Each y gets dydx (3) Collect terms (4) Solve for dydx

Ex: x² + y² = 25 → 2x + 2y(dydx) = 0 → dydx = −xy
 

Related Rates

(1) Equation relating variables (2) Differentiate w.r.t. t (3) Plug in known (4) Solve for unknown rate

Ex (Cube): V = s³, dVdt = 48, s = 15
dVdt = 3s²dsdt → 48 = 675dsdt → dsdt = 0.0711
Ex (Cylinder): V = πr²h, r = 30 (constant), dhdt = 560
dVdt = πr²dhdt = π(30)²(560) = 1,583,362.7
 
 

UNIT 3: THEORY OF DERIVATIVES

 

Critical Points

f'(x) = 0   or   f'(x) undefined
 

First Derivative Test

Test sign of f'(x) around critical point c:

+ + + | - - - → local MAX
- - - | + + + → local MIN
+ + + | + + + → neither

 

Second Derivative Test

At critical point c where f'(c) = 0:

f''(c) > 0 → local MIN (∪) | f''(c) < 0 → local MAX (∩) | f''(c) = 0 → inconclusive

Ex: f(x) = 2(x−4)²x+6 → Find f'(x) = 0, then test f''(critical points)
 

Concavity

f''(x) > 0 → concave up ∪ | f''(x) < 0 → concave down ∩

Inflection: f''(x) = 0 AND sign change

 

L'Hôpital's Rule

Use when: 00 or ∞∞

lim f(x)g(x) = lim f'(x)g'(x)

Differentiate top and bottom SEPARATELY

Ex: limx→1 3ln(x)x−1 = 00 → limx→1 3/x1 = 3
Ex: limx→12 [964x²−64x+192 − x4x−48] → Combine, get 00, L'H → −0.625
 

Optimization

(1) Identify max/min (2) Equation Q(x) (3) Constraints → one variable (4) Q'(x) = 0 (5) Test

Ex: Max area under parabola: A = 18x − 2x³ → A' = 18 − 6x² = 0 → x = √3
 
 

UNIT 4: INTEGRALS

 

Basic Integration

∫ xn dx = xn+1n+1 + C  (n ≠ −1)

∫ 1 dx = x + C | ∫ x dx = x²2 + C | ∫ 1x dx = ln|x| + C | ∫ ex dx = ex + C

∫ sin(x) dx = −cos(x) + C | ∫ cos(x) dx = sin(x) + C | ∫ sec²(x) dx = tan(x) + C

 

Definite Integrals

∫ab f(x) dx = F(b) − F(a)
Ex: ∫02 x² dx = [x³3]02 = 83 − 0 = 83
 

U-Substitution

(1) u = inside (2) du = derivative·dx (3) Rewrite in u (4) Integrate (5) Substitute back

Ex: ∫ 2x·ex² dx → u = x², du = 2x dx → ∫ eu du = eu + C = ex² + C
 

Area Under Curve

A = ∫ab f(x) dx  (when f(x) ≥ 0)
 

Area Between Curves

A = ∫ab [f(x) − g(x)] dx  (top − bottom)

(1) Find intersections (2) Determine which is on top (3) Integrate

Ex: Between y = x and y = x² from 0 to 1
A = ∫01 (x − x²) dx = [x²2 − x³3]01 = 12 − 13 = 16
 

Work

W = ∫ab F(x) dx

Basic: Work = Force × Distance (when force is constant)

Units: 1 Newton = 1 kg·m/s² | Work in Joules = Newton·meters

Springs (Hooke's Law): F(x) = kx, where k = spring constant, x = displacement

 

Volume - Disk Method

V = π∫ab [f(x)]² dx  (around x-axis)
Ex: y = √x from 0 to 4, around x-axis
V = π∫04 (√x)² dx = π∫04 x dx = π[x²2]04 = 8π
 

Volume - Washer Method

V = π∫ab [R(x)² − r(x)²] dx

R = outer radius, r = inner radius

Ex: Between y = x and y = x², around x-axis
R(x) = x, r(x) = x²
V = π∫01 (x² − x⁴) dx = π[x³3 − x⁵5]01 = 2π15
 

Volume - Shell Method

V = 2π∫ab x·f(x) dx  (around y-axis)

radius × height

Ex: y = x² from 0 to 2, around y-axis
radius = x, height = x²
V = 2π∫02 x·x² dx = 2π∫02 x³ dx = 2π[x⁴4]02 = 8π
 

Rotating Around x = k or y = k

Around x = k: Radius = |x − k|

Around y = k: Radius = |y − k|

Adjust radius in formulas accordingly
