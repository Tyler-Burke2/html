Project Overview

Star Typer is an engaging space-themed typing game that combines fast-paced gameplay with an extensive progression system. Players type words as they fall from space, earning currency to purchase upgrades, unlock skills, and increase their combo multipliers for higher scores.

Gameplay Demo

Key Features

  • Dynamic typing gameplay with falling words in a space theme
  • Comprehensive upgrade system with shop and permanent skills
  • Combo multiplier system that rewards consistent accuracy
  • Multiple word banks with varying difficulty levels
  • Full save/load system with export/import functionality
  • Auto-save feature to preserve progress
  • Golden word chances and special word types (sentences)
  • Detailed stats tracking for player performance
  • Background music and immersive sound design

Technologies Used

HTML5 CSS3 JavaScript ES6+ Local Storage API Canvas/Animation

Challenges & Solutions

The biggest challenges were file management and the complex math behind the upgrade system. Managing save/load functionality required careful handling of data serialization and ensuring backwards compatibility as features were added. The upgrade math needed to be carefully balanced to create a satisfying progression curve - making upgrades feel impactful while maintaining long-term playability. I developed formulas for exponential scaling costs and multiplicative effects that keep the game engaging throughout extended play sessions.

Technical Implementation

This solo project spans approximately 10,000 lines of code, built over the course of one month. The game uses vanilla JavaScript for all game logic, with careful attention to performance optimization for smooth animations. The save system leverages browser Local Storage to persist player progress, with additional export/import features for backup and transfer between devices.

Design & User Experience

The interface features a clean, space-themed design with intuitive controls. Players can type anywhere on the screen and press Enter to submit, with helpful keyboard shortcuts (Backspace to edit, Escape to clear). The responsive layout ensures the game works seamlessly across different screen sizes.

Results & Learning

Star Typer demonstrates my ability to build complex, feature-rich web applications from scratch. The project strengthened my skills in game development, data persistence, mathematical modeling for game balance, and creating engaging user experiences with pure JavaScript.
